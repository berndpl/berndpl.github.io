# plontsch.de

`plontsch.de` is the canonical About page in the Website family:

- `plontsch.de`
- `notes.plontsch.de`
- the published Project Websites listed in the family roster

The root page and 404 use local copies of the Notes-owned family assets in `assets/`:

- `family-header.css` owns content width, responsive insets, transparent header geometry, and shared structural metrics
- `family-typography.css` owns the local Inconsolata / JetBrains Mono type system
- `family-theme.css` and `family-theme.js` own appearance and cross-domain selection
- `family-footer.css`, `family-footer.js`, and `family-actions.css` own shared controls and interaction

Update the canonical sources in `notes-plontsch` and the family contract first, then refresh this
site and every published Project Website together.
